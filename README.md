# Módulo de Venta

Sistema básico de ventas desarrollado con Django.

## Funcionalidades

- Gestión de productos
- Gestión de clientes
- Registro de ventas
- Panel administrativo

## Requisitos

- Python 3.10 o superior
- Git
- Git Bash en Windows

```bash
git clone https://github.com/ElKhasta/Modulo-de-venta.git
cd Modulo-de-venta/backend
